A simple water mark component based on react.

there is a demo below:

``` js
	import WaterMark from 'water-mark-react';

	class Demo extends Component {
		render() {
			return <WaterMark
						info={{
                            width: 100,
                            height: 100,
                            text: 'demo'
                        }}
                    />
		}
	}

```
